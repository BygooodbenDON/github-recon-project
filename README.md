# GitHub 偵察機 🚀

這是一個專為初學者設計的 GitHub 使用者搜尋工具，使用 Vue 3 打造。

## 🎓 我的學習紀錄

### 💡 我與 AI 導師的問答筆記

#### 1. 如果我在輸入框打字、按下按鈕，到照片出現，這中間經過了哪些關卡？
- **AI 導師的比喻**：這是一場「包裹傳遞遊戲」。
- **簡短說明**：你打的字會存進**「神奇盒子 (Ref)」**，按按鈕是吹響**「小喇叭 (@click)」**，接著派出**「外送員 (Fetch)」**去 GitHub 領包裹，最後撥開**「隱身開關 (v-if)」**讓照片現身。

#### 2. 你是用哪一行指令來控制這行字「什麼時候該出現」的？
- **AI 導師的比喻**：使用「隱身開關」。
- **簡短說明**：就是程式碼裡的 `v-if`。它會根據盒子裡的開關狀態（true 或 false），決定該行字要顯示還是隱身。

#### 3. 資料是怎麼查到的？還有旋轉燈怎麼做的？
- **AI 導師的比喻**：使用「專屬地址」與「自動旋轉小風車」。
- **簡短說明**：資料是給外送員一個 **URL 地址**去雲端圖書館拿的。旋轉燈則是利用一個**「外送中開關」**，搭配 CSS 讓圓圈像小風車一樣不停旋轉。

#### 4. 如何讓多個作品在畫面上排排站？
- **AI 導師的比喻**：使用「排排站指令」。
- **簡短說明**：就是 `v-for` 指令。它像領隊帶隊一樣，把作品袋子裡的成員一個接一個請出來，整齊地印在網頁畫面上。
