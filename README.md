# GitHub 偵察機 🚀

這是一個專為初學者設計的 GitHub 使用者搜尋工具，使用 Vue 3 打造。

## 🎓 我的學習紀錄

### 1. 我問了 AI 什麼問題來搞懂程式碼？
為了搞清楚網頁是怎麼運作的，我問了 AI：
- 「如果我在輸入框打字、按下按鈕，到照片出現，這中間經過了哪些關卡？」
- 「你是用哪一行指令來控制這行字『什麼時候該出現』的？」

### 2. AI 給了我什麼「比喻」讓我聽懂了？
AI 導師用了非常生活化的比喻，讓我這個只懂 HTML 的人也能聽懂：
- **神奇盒子 (Ref)**：用來存資料的地方，內容變了畫面會自動變身。
- **隱身開關 (v-if)**：就像電燈開關，決定東西要不要顯示。
- **點擊小喇叭 (@click)**：按鈕按下去時發出的信號。
- **外送員 (Fetch)**：幫我跑腿去 GitHub 拿資料的人。
